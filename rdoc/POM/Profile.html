Profile

Profile store ancillary project metadata such as title, summary, list of authors, etc.

Constants

RE_EMAIL

Attributes

name[R]

Project’s package name. The entry is required and must not contain spaces or puncuation.

title[RW]

Title of package (this defaults to project name capitalized).

summary[RW]

A one-line brief description.

description[RW]

Detailed description.

suite[RW]

Name of the user-account or master-project to which this project belongs. The suite name defaults to the project name if no entry is given. This is also aliased as #.

contact[RW]

Official contact for this project. This is typically a name and email address.

created[RW]

The date the project was started.

copyright[RW]

Copyright notice. Eg. “Copyright © 2009 Thomas Sawyer“

license[RW]

License.

authors[RW]

List of authors.
