== Generate POM Metadata from README

We generate the metadata from the README using the pom command.

  system "pom init fixture/README"

Now lets verify the metadata was extracted as expected.

  require 'pom/metadata'

  metadata = POM::Metadata.load

  metadata.title.assert == "MyApp"
  metadata.description.assert == "This is a description of the\nfake project."
  metadata.license.assert == "GPL"

Clean-up the generated directory.

  FileUtils.rm_r('meta')
  FileUtils.rm_r('.cache') if File.exist?('.cache')
